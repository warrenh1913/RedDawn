# RedDawn
Welcome to RedDawn

### Objective
Get the key, eliminate stalin, escape from the facility

### GamePlay
Click left mouse button to shoot, use the top 1,2,3 keys to switch weapons once you find and collect them thoughout the map. Use WASD keys to move.

# Instalation Guide
We highly recomend using windows to play the game. Macbooks will require some command line inputs to allow the app to run.

### Windows
Download the winodws version of the program from the github release. Then extract the zip file and double click on RedDawn.exe

### Macbook
Download the Mac version of the program from the github release. Then extract the zip file and open a terminal in the folder. Then run the following commands:

chmod -R +x RedDawnMac.app

xattr -d com.apple.quarantine RedDawnMac.app

then run it using,

open RedDawnMac.app

### Linux
Download the linux version of the program from the github release. Then extract the tarball using:

tar -xf RedDawnLinux.tar

get into the directory using cd RedDawnLinux and run the program using ./RedDawn.x86_64

if it says permission denied use the command:

chmod 777 RedDawn.x86_64 

and then try running it agian
